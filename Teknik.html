Tekniken bakom webben

Den Teknik som bygger upp webben kan man främst dela upp i tre delar, serverteknik, klientteknik och koden som bygger upp Webbsidorna.

Servertekniken, även kallat hosting, bygger på att det finns en webbserver, som är uppkopplad till internet, som lagrar information och distribuera informationen som webbsidor till klienter. Webbservern tar först emot en förfrågan från en klient och skickar sedan tillbaka den information som det frågades om. Självaste ordet webbserver kan både referera till fysiska datorerna och mjukvaran som används.

Klient-tekniken är webbläsaren som används av klienten. En webbläsare är en mjukvara som hämtar informationen från webbservern och som sedan tolkar denna information som ges tillbaka så att den är förståelig för klienten. I denna informationen finns även mjukvara, som kallas script, som körs direkt på klientens webbläsare vilket minskar arbetet som webbservern måste utföra.

Koden som bygger upp webbsidor består främst av språken html, css och ett skriptspråk t.ex javascript. Html är en förkortning av Hyper Text Markup Language och vad som kallas ett märkspråk eller sidbeskrivningsspråk. Html utgör strukturen för webbsidor och innehåller all information som ska visa fram för klienten. Detta utgör allt från rubriker, styckeindelningar, paragrafer m.m. CSS (Cascading Style Sheets) beskriver sidans utseende och design, alltså hur den färdig webbsidan kommer att se ut bestäms utav CSS. Sist så används skriptspråk såsom Javascript som är en typ av simplare programmeringsspråk skapat för utveckling av webbsidor. Dessa skriptspråk gör webbsidor mer interaktiva för användaren.
